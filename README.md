# 2. markdown 문법


## 제목(Headers)
 - 글씨 크기를 변환한다. ` # `의 갯수가 늘어날수록 글씨가 작아진다.

>입력
```
 # 가장 큰 제목 (H1)
 ## 두 번째 수준 제목 (H2)
 ### 세 번째 수준 제목 (H3)
 #### 네 번째 수준 제목 (H4)
 ##### 다섯 번째 수준 제목 (H5)
 ###### 여섯 번째 수준 제목 (H6)
```

>출력
>> # 가장 큰 제목 (H1)
>> ## 두 번째 수준 제목 (H2)
>> ### 세 번째 수준 제목 (H3)
>> #### 네 번째 수준 제목 (H4)
>> ##### 다섯 번째 수준 제목 (H5)
>> ###### 여섯 번째 수준 제목 (H6)


## 줄바꿈
- 줄을 바꿀 때는 엔터를 두번, 1줄 띄어준다.

>입력
```
줄을 바꾼다.

줄을 바꾼다.
```

>출력

>>줄을 바꾼다.

>>줄을 바꾼다.

 ## 코드블럭(code Blocks)
 - 코드를 감싸서 코드를 그대로 보여주고 싶을 때 사용한다. 짧은 코드는 \` 를 사용한다.
 
 >입력
 ```
 `<table><tr><td></td></tr></table>`
 ```

>출력
>> `<table><tr><td></td></tr></table>`

- 줄이 한 줄이 아닐 때는 시작 줄과 끝 줄에 \` 나  ~기호를 세개 쓴다.

>입력
``` 
void f()
    printf(%s,“이것은 c 코드 입니다”);
}
 ```

>출력

```
void f()
    printf(%s,“이것은 c 코드 입니다”);
}
```


## 강조(Emphasis)
- 기울여 쓰기(italic)는 ` * ` 또는 ` _ `로 감싸고 굵게 쓰기(bold)는 ` ** ` 또는 ` __ `로 감싼다.

>입력
```
*기울여 쓰기*
_기울여 쓰기_

**굵게 쓰기**
__굵게 쓰기__
```

>출력
>>*기울여 쓰기*\
>>_기울여 쓰기_

>>**굵게 쓰기**\
>>__굵게 쓰기__


## 수평선(Horizontal Rules)
- ` - `나 ` * `, 또는 ` _ `을 3개 이상 작성하면 수평선으로 출력된다.(단, ` - `을 사용할 경우 header로 인식할 수 있으니 전 라인은 비워야 한다.

>입력
```
---
***
___
*****
```

>출력
>>---
>>***
>>___
>>*****


## 링크(Links)
- 인라인 링크 : 링크의 이름과 주소를 설정하여 이름을 누르면 링크로 이동한다. 형식은 ` [링크](http://주소) ` 이다.

>입력
```
[구글](http://google.com)
```

>출력
>>[구글](http://google.com)


- 참조 링크 : 사전에 설정한 이름을 불러서 사용한다. `[example][a] ` 는 eample라는 이름을 a 로 설정해 eample을 많이 쓰는 곳에 편리하게 쓰인다. 

>입력
```
[google][a]
[naver][b]
[daum][]  // 만약 링크되는 문구 자체가 키워드라면 대괄호를 내용없이 한 번만 더 붙여주면 된다.

[a]:http://google.com
[b]:http://naver.com
[daum]:http://daum.net
```

>출력
>>[google][a]\
>>[naver][b]\
>>[daum][]

>>[a]:http://google.com\
>>[b]:http://naver.com\
>>[daum]:http://daum.net

- URL 링크 : 별도의 이름 없이 ` <  > ` 안에 주소를 넣으면 주소가 링크로 출력된다. 이메일도 가능하다.

>입력
```
<http://google.com>

<oja5068@gmail.com>
```

>출력
>><http://google.com>

>><oja5068@gmail.com>


## 리스트(Lists)
- 기호로 표현하려면 ` *, +, -`중 아무거나 쓰면 된다. 주의해야할 점은 숫자든 기호든 한칸 띄어야한다.

>입력
```
* 시험
* 과제
* 출석
```

>출력
>> * 시험
>> + 과제
>> - 출석

- 숫자로 표현하려면 '숫자. ' 형식으로 쓰면 된다. 숫자 다음 점 찍고 쓴다. 적는 숫자랑 상관없이 순서대로 번호가 매겨진다.

>입력
```
1. 시험
2. 과제
1. 출석
```

>출력
>>1. 시험
>>2. 과제
>>1. 출석


## 테이블(Tables)
- 기호는 ` -, :, | `를 사용한다. 정렬하는 기준은 ` : `의 위치이고 가운데 정렬은 ` <center> `태그를 사용해야한다.

>입력
```
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
|:--------|:--------:|--------:|
| 내용 11 | <center>내용 12</center> | 내용 13 |
| 내용 21 | <center>내용 22</center> | 내용 23 |
```

>출력

| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
|:--------|:--------:|--------:|
| 내용 11 | <center>내용 12</center> | 내용 13 |
| 내용 21 | <center>내용 22</center> | 내용 23 |


## 이미지 삽입(Images)
- 이미지 삽입은 URL경로를 전체를 써도 되고, title 지정되어 있으면 같은 서버 경로에서 호출해도 된다. 쉽게 말해서 링크처럼 인라인, 링크, 참조 형식이 있다.

>입력
```
![alt text](/test.png) //인라인 이미지

![alt text](image_URL) //링크 이미지

![alt text][1]

[1]:/test.png //참조 이미지
```

>출력
>>![alt text](/test.png)

>>![alt text](image_URL)

>>![alt text][1]

>>[1]:/test.png
